# Examples for Team Building Facilitator

This file provides concrete usage examples and scenarios demonstrating this prompt in action.

---

### Scenario: Software Development Team (8 members, 3 months old)

**Stage 1: Assessment**
- Stage: Storming (conflicts emerging, role confusion)
- Challenges: Communication breakdowns, unclear responsibilities
- Goal: Move to Norming within 2 months

**Stage 2: Stage Identification**
- Confirmed: Storming stage
- Evidence: Conflicts over code reviews, unclear decision-making
- Intervention: Focus on communication and role clarity

**Stage 3: Trust Building**
- Week 1: Personal Histories Exercise
- Week 2: Team Charter (defined communication norms)
- Week 3: Vulnerability session on project challenges
- Ongoing: Weekly weather check-ins

**Stage 4: Roles and Collaboration**
- Belbin assessment revealed: 3 Shapers, no Completer-Finisher
- Adjusted: Hired CF-strong contractor, coached Shapers on collaboration
- Created: Decision-making protocol (consensus for architecture, async for minor)

**Stage 5: Conflict Resolution**
- Trained team in NVC
- Implemented bi-weekly retrospectives
- Resolved code review conflicts through pairing sessions
- Result: 30% faster delivery, 80% team satisfaction (up from 50%)
