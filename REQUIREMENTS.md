## 1. プロジェクト概要

### 1.1 目的
対話型Agentic AIプロンプトのテンプレートライブラリを構築し、ユーザーが目的を入力すると、AIが目的を明確化するための対話を設計し、対話を通じて得られたコンテキストと専門知識・フレームワークを活用してメタプロンプトを生成できる仕組みを提供する。

### 1.2 スコープ
- マークダウンベースのプロンプトテンプレート集の作成
- YAMLマニフェストによるテンプレート管理
- 幅広い専門領域をカバーするサンプルプロンプトの提供
- ドキュメントとして参照・活用可能な形式での提供

### 1.3 対象ユーザー
- AIプロンプトエンジニア
- ビジネスコンサルタント
- ソフトウェア開発者・設計者
- 教育関係者
- 研究者
- その他、AIを活用した業務改善を目指す専門家

---

## 2. 機能要件

### 2.1 プロンプトテンプレート
各プロンプトテンプレートは以下の要素を含む:

#### 2.1.1 構造
- **YAMLフロントマター**: メタデータ管理
  - id: プロンプトの一意識別子
  - category: カテゴリ分類
  - frameworks: 活用するフレームワーク・理論のリスト
  - dialogue_stages: 対話ステージ数
  - version: バージョン番号
  - tags: 検索・分類用タグ
  
- **本文セクション**:
  - 役割定義
  - 対話フロー（ステージごとの詳細）
  - 活用する専門知識・フレームワーク
  - 出力形式
  - 使用例（オプション）

#### 2.1.2 対話フロー
- プロンプトの種類によって対話ステージ数は可変
- 一般的な流れ:
  1. 目的の明確化
  2. コンテキスト情報の収集
  3. 専門知識・フレームワークの適用
  4. メタプロンプトの生成

### 2.2 マニフェスト管理
- YAML形式でのメタデータ管理
- プロンプト一覧の階層的な整理
- カテゴリ別、タグ別の分類
- バージョン管理

## 4. 技術仕様

### 4.1 実装形態
- **形式**: マークダウンベースのドキュメント
- **管理**: YAMLマニフェスト
- **配置**: `/home/nahisaho/GitHub/ouranos-agentic-library/`

### 4.2 ディレクトリ構成
```
ouranos-agentic-library/
├── manifest.yml              # プロンプト全体のメタデータ管理
├── README.md                 # プロジェクト説明、使い方ガイド
├── LICENSE                   # CC BY-NC 4.0
├── prompts/                  # プロンプトテンプレート
│   ├── general/              # 汎用
│   ├── business-management/  # ビジネス・経営
│   ├── design-development/   # デザイン・開発
│   ├── hr-organization/      # 人材・組織
│   ├── education/            # 教育
│   ├── research/             # 研究
│   ├── document-creation/    # 文書作成
│   ├── social-policy/        # 社会・政策
│   ├── communication/        # コミュニケーション
│   ├── innovation-transformation/ # イノベーション・変革
│   └── specialized-domains/  # 専門領域
└── docs/                     # 追加ドキュメント
    ├── usage-guide.md        # 使い方ガイド
    ├── contribution.md       # 貢献ガイドライン
```

### 4.3 ファイル命名規則
- プロンプトファイル: `{domain}-{specific-name}.md`
- 例: `business-strategy-consultant.md`, `software-architect.md`
- 小文字、ハイフン区切り
