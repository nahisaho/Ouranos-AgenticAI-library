## 使用方法

各プロンプトファイルは `prompts/copilot/prompts/<カテゴリ>/<ファイル名>` に配置されています。

詳細なフレームワーク、例、メソドロジーは各プロンプトに対応するRAGファイル (`prompts/copilot/rag/<カテゴリ>/<プロンプトID>/`) に格納されています。

### プロンプトの選び方

1. **カテゴリで探す**: 上記の目次から該当するカテゴリを選択
2. **用途で探す**: 表の「用途」列で具体的な支援内容を確認
3. **IDで参照**: プロンプトIDを使用してファイルを直接参照
